# Block_chaining

## 개발 목적
암호화를 통해 동영상 URL, 동영상 Contents 의 보안성 향상을 목적으로 한다.

## 개발 내용
SEED 128 을 통한 File(Byte) 암호화, 복호화
SEED 128 을 통한 String 암호화, 복호화
